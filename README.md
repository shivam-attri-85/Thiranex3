# Thiranex3

A single-page TaskFlow to-do list application built with HTML, CSS, and vanilla JavaScript.

## Features

- Create, edit, complete, and delete tasks
- Filter tasks by All, Active, and Completed
- Persist task data with `window.localStorage`
- Use delegated DOM events for task actions

## Files

- `TaskFlow.html` - the full application in one file

## Run

Open `TaskFlow.html` in a browser to use the app locally.
